# CAPY graphics

## Overview

CAPY graphics aims to handle the graphical and connotative development of CAPY's branding, style, and identity. 

## Includes:

- Logos
- Colors
- Fonts
- Images
- Illustrations
- Icons
- Mission
- Vision
- Values
- Voice
- Tone
- Formatting preferences
- Copy guidelines
- Brand templates
- Resources and points of contact
